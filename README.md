# HTML 5 task (the fifth task)
The task is:

```
Draw a car with CSS using as little HTML elements as possible.
It could be a car, truck or even a bicycle, thing is it must look like a vehicle.

The vehicle's width must be 200px.

On the page load, the vehicle starts its way from the leftmost of the screen and stops on 50px before the rightmost point.

You can add a background (road/sun/home/tree) if you want
```
